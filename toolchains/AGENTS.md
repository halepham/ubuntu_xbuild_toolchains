# Workspace AI Guidance

This repository contains workspace-specific GitHub Copilot guidance and
skills.

Before acting on a task in this repository:

1. Read `.github/copilot-instructions.md`.
2. Use its skill router to decide whether a workspace skill applies.
3. If a skill applies, read only the matching
   `.github/skills/<skill-name>/SKILL.md`.
4. Follow that skill's workflow before running commands or editing files.

Prefer the workspace-specific guidance over generic ROS 2, build, deploy,
debug, test, diagram, or git-commit workflows.
